# Fake News Detection using Machine Learning

## Overview
This project predicts whether a news article is **Fake** or **Real** using Machine Learning.

## Features
- Data preprocessing
- Text cleaning
- TF-IDF Vectorization
- Logistic Regression
- Manual news prediction

## Technologies Used
- Python
- Pandas
- NumPy
- NLTK
- Scikit-learn
- Google Colab

## Dataset
The dataset is not included because of GitHub file size limits.

Download it from:
https://www.kaggle.com/datasets/clmentbisaillon/fake-and-real-news-dataset

Place **Fake.csv** and **True.csv** in the project folder before running the notebook.

## How to Run

1. Install the required libraries.
2. Download the dataset.
3. Place `Fake.csv` and `True.csv` in the project folder.
4. Open `Fake_News_Detection.ipynb`.
5. Run all cells.
6. Enter a news article to classify it.

## Project Output
The model predicts whether the news is:
- Fake News
- Real News
